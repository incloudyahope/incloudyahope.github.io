# Modern CV (React + Vite)

Template modern untuk menampilkan CV dan preview PDF.

Quick start

```bash
cd cv-site
npm install
npm run dev
```

Usage

- Copy your PDF `Mohamad Eko P CV 2025_Juni (1).pdf` into `public` and rename it to `resume.pdf`.
- Open `http://localhost:5173` and lihat preview serta download.

Next steps (optional)

- Replace content in `src/components/*` dengan data nyata dari CV.
- Integrate `pdfjs-dist` jika ingin ekstrak teks otomatis.
